# Pokémon Web Demo

Slice 1 del apply SDD: shell visible, catálogo local CSV y mano inicial renderizada en React.

## Correr localmente

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```

## Dataset

El catálogo local vive en `public/data/pokemon_cards_gen1.csv` y se filtra por `categoria = Pokémon`.
